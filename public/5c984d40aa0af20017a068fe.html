スクボで読む

Chrome の DevTools で任意のリクエストをブロックする

リクエストがエラった時などを再現するために意図的にリクエストを失敗させたいことがある
Chrome の DevTools で Network タブから当該リクエストをフルパスかドメイン単位でブロックすることができる
ブロックされるリクエストは Request blocking というタブに一覧になっている
こうしておくと JS ランドには通信の失敗としてエラーが上がってくるので便利
