TrueNAS の Dataset を配信する
TrueNAS に Dataset の中身を http で吐けるみたいな機能があったらラクだなーと調べてみたが、特にないらしい
マウントしてリバースプロキシ立てればよくない？みたいな感じだったので、まあそうなんか、っていう
というわけで、手っ取り早くリバースプロキシたてるなら〜ということで
version: "3.5"
services:
nginx:
image: nginx
ports:
- 80:80
volumes:
- /mnt/data:/usr/share/nginx/html
こういう docker-compose.yml を書いて up しとけばよい
まあ宅内ネットだし一旦認証とかは考えないものとする
