古のChrome拡張をvite化する
Scrapbox にページをブチ込むための拡張 Scboloo の私家版をたま〜に手を入れたりなんだりしている
2017 年あたり製ぽいのでかれこれ5年か...
ちょこっとだけ変更を入れたかったんだけど、webpack だの babel だのもう関わりたくないので vite に置き換えてしまうことにした
https://github.com/e-jigsaw/scboloo/commit/1ee401a89ab29e22b980d18c730e7e3352eab39c
最近 pnpm 使ってるので pnpm に変えて、ビルドまわりを vite にした
then-chrome という chrome の API を promise にラップするやつがもう動かなくなってて、Manifest V3 にしたら(?)なのかわからないがもう promise を返すようになっているので不要と判断して捨てた
実際の拡張のコード自体はシンプルなので、依存パッケージもなくシンプルな拡張になった
crxjs にすると background が自動的に service worker になるようで、ちょっと挙動が心配だったけど特段困ることはなかった
