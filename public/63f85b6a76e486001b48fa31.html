Proxmox に Ubuntu の VM をつくる
Ubuntu のダウンロードページから Server の LTS の ISO をダウンロードする
ISO はホストにアップロードしておかないといけないみたい
local の ISO Images にアップロードしておく
Create VM から適当に設定をポチポチする
https://zenn.dev/northeggman/articles/d7b29c4e2cd7c7 を参考にした
GUI はいらないのでそのあたりは全部スキップ
こういう設定にしてみた
メモリカツカツでも困るかなとおもって半分ぐらい割り当ててしまったけど減らしてもいいかも
そしたら Web の GUI から noVNC で接続できるので、Ubuntu のインストーラのガイドに従って設定をポチポチしていく
むちゃくちゃ久しぶりに Ubuntu のインストールやったけど、最終セクションあたりで Docker とか名の知れてるミドルをインストールする導線があったので、とりあえず Docker はここでインストールした
再起動前後らへんでハードウェアのところからCDドライブに入ってることになっている Ubuntu のインストーラを排出しておく
