Perfume の GAME は改めて名盤

Perfume が Perfume The Best "P Cubed" を出したのに合わせて過去の盤も一気にサブスク解禁された
P Cubed を聴き返していると、やっぱりヘビロテしてた盤の曲はグッとくるし、前後の曲なんかも一緒に聴きたくなってしまったりする
特に Perfume に対しては懐古っぽいことをしたくないという感情はあるんだけど、やっぱり GAME なんかは思い入れが強すぎて引っ張られる感じがある
⊿ や JPN らへんも思い入れがあるけど、やっぱり GAME は格別
あとはライブをみにいけた LEVEL3 もかな
