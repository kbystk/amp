antd 4系にマイグレート
結構前にやったんだけどメモるのを忘れていた
基本的には https://next.ant.design/docs/react/migration-v4 をみればよい
使ってる範囲で主にやったのは dropdownMatchSelectWidth は boolean じゃなくて number になったっぽい
TimePicker の defaultOpenValue は defaultValue になった
moment と dayjs の話が書いてあって、置き換えないといけないのかとおもったけど moment で十分そう
Icon まわりがごっそり変わって @ant-design/icons から import してくるようになった
Table が ColumnProps から ColumnsType に変わった
細かいけど autosize が autoSize に変わった
今のところ問題なくいけてる
ちなみに、同じタイミングで yarn を berry にしようとしたんだけどこの antd まわりのパッケージの解決にどうしても失敗するので1系に戻した
