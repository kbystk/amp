上京 - diary.jgs.me

ついに今日 上京 する。今日上京って 韻が踏める 。
昨日はアレの発表があって、久々に口の中がパッサパサになる程度にはほどよく緊張感をもって臨めた。 発表資料 はこれまでに自分がつくった資料のなかで 最悪なルックス に書き換えることを余儀なくされたが、こういう場ではそれがいいのだろう。練習では一度も褒められたことがなかったが、先週の フィードバック を基に週末にゴッと資料を直した甲斐あってお褒めの言葉をいただいた。
発表のあとは、 成果物 や メモ をまとめたりした。 シベリアのマシン で動くかどうか確認するために、 計算機室のマシン を叩いていたら、かなり強そうな .zshrc と .vimrc が出てきて驚いた。とはいえ、 node が0.8系が入ってるし、 python は2.4系が入ってるしで 寿命 が縮みそうだった。 git は入ってなかったし。手元でnodeの0.10系とpythonの2.7系をビルドして事無きを得た。
Github に公開するぐらいの README を用意していたのだけれど、それだと不十分で、最終的にかなり過激なぐらいにしつこく文章を書くことになった。それもまあいいよ。全て終わったからね。 シベリアの鍵 を返した。
今日は朝から 引越し や 退去 のなんやかんやがあるので早寝したら早起きしすぎた。余裕をもって出ていこうとおもう。
