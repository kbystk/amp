久しぶりにラズパイ
今さら感あるかもしれないが、やっぱり部屋にこもって仕事してると二酸化炭素濃度やばいのでは？という疑念が
適当にセンサを買おうと探してみたが、画面に表示だけするやつは数あれどセンサの値を引っ張ってきて記録しておく、というやつは全然見つからなかった
仕方ないのでマジのセンサだけ買ってなんらか電子工作するしかねえか...とおもい、そっちも調べてると M5Stack? M5Stick? というやつにつなげたりするやつが出てきたりした
こういう時ぐらい他の言語書けよという気がしないでもないけど、慣れてるツールでコードを書きたい欲が出てきてしまって、どうにか Raspberry Pi でやれないかなと
大昔に買ったラズパイ3が引っ越しだかのタイミングでぶっ壊れてしまっていて、復旧できるか試した
https://dev.classmethod.jp/articles/restore-the-raspberry-pi-4-model-b-os-destroyed-by-a-biginner/
によると、不意の電源抜き差しで SD がぶっ壊れるらしい
ひとまず SD に OS を焼き直してみることに
https://www.raspberrypi.com/software/
なんと SD 焼くアプリを公式が配布してて、これでぱぱっと焼くことに成功した
SD(microも)のスロットがついたデバイスなどなかったので、これも大昔に使っていた microSD => USB-A のコネクタに USB-A => USB-C のコネクタをかませないと macbook につながらなかったのだが、さすがにタワーすぎるせいか mac だと認識されなかった
仕方ないので母艦の windows に USB-A で接続したら、認識はするものの、そもそも SD がぶっ壊れてそうだった
コネクタがイカれてるのか、本体がイカれてるのかわからなかったので、転がってた8GB の SD を挿してみると無事に認識されたのでこのままこいつでいくことにした
容量半分になってしまったので些か不安ではある
https://qiita.com/c60evaporator/items/ebe9c6e8a445fed859dc
linux でもろもろ整えるの毎回やっては忘れてしまうので、これを参考に整えた
初手だと GUI が上がってしまって、適当なマウスがなくて詰んだのだが、キーボードで ctrl+alt+t でターミナルが立ち上がるってことだったので、これでターミナルを開いて raspi-config を実行して CUI をデフォルトにしたり、ssh を enable にしたりした
最近はもっぱら volta 使ってるので volta 使えるかな〜とおもったが、volta は x86 しかダメらしい
nvm 試してみたらちゃんと armv7l のバイナリを引いてきてくれてさすがだなと
普通に node@16 動いた
ちなみに、deno も動いたりしないかな〜とおもって curl -fsSL https://deno.land/x/install/install.sh | sh したら x86? のバイナリが入ってきただけで使えなかった
しょうがないので rust の環境を入れた
rust は curl https://sh.rustup.rs -sSf | sh だけで入って便利
んで cargo install deno --locked を試したんだけど、rustry_v8 のコンパイルで失敗するっぽい
ちょこっと調べて https://github.com/LukeChannings/deno-arm64 で配布されてるバイナリを試してみたけど動かず
今回は deno はサブなので一旦スルーすることにした
