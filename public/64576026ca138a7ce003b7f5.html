Apex Legends その1
Sons Of The Forest にキレた結果、トリガーハッピーなゲームやりたいなーと、勢いで Apex ってやつをやってみるか！と
スプラ3で年末ぐらいから固定でがんばってたのがひと段落して、ちょっと別ゲーやりたいなということもあり
ちょうどフレにダイアの人がいるので、立ち回りとかいろいろ教わりつつ
タルコフ、スプラ3に続いてシーズン終了間際に突然異常な時間プレイし続けるムーブをしてしまった
まあこれやっとくと新シリーズからやってみるか～よりは多少はマシにスタートダッシュをきれるのでまあまあアリかなとはおもっている
ちなみにはじめてチャンピオンになった時のリザルトはこんな感じで、とにかくライフラインの人がTUEEEEだった、マジになんもしてない
ルーキーからあげていって、ブロ1まできた
チャンピオン2回目
初回よりは多少アシストはできたけど、結局のところ味方がちょーつえーから勝ったみたいな試合だった
とにかく今は全てにおいて経験値(PS)が足りてない状態って感じなのでまだまだ伸びしろしかねえなあ!!とポジティブに1戦1戦たたかっている
課題に感じてるのは
ジャンプ
基本的にはかぶんないで降りたい気持ち
敢えてかぶせてキル稼ぐという戦術もあるみたいだけど、それは無理
ジャンプマスターでかぶんないで降りるのが難しい
フレに「ルール#1は後半に降りることを徹底する」だと教わって、最初はこれをやるだけで自動的に半分から上状態が確定できてたんだけど、自分のランクが上がっていく毎に厳しくなってきていて、単純に待っててもキツいという印象になってきた
かといってうまいところにガイドできるかというとむずい
立ち回り
味方が交戦はじめてるのに、自分がちょっと遠くにいてて...がそこそこある
場所とかシチュエーションによって、相手がこの辺からきそうだなーみたいな読みが薄いので、後手に回ってしまう
詰める判断が難しい
基本ブラッドハウンドをメインにしてて、詰める時はウルト発動してガンダッシュで詰めるが恐らく正解だとおもってるんだけど、それを味方と合わせるのが結構ムズい
「これはもう詰めてやりきるしかないっしょ！」とおもってウルト吐いても味方全然ついてこないとただ凸った人になっちゃうという...
味方が攻め起点つくってくれるタイプの人だと、合わせでウルト切って～...ができるんだけど、そこまで何回もワークした感はまだない
ルート
フレからも「もう少し漁り速度あげないとダメ」と言われているので、ちゃきちゃき判断していかないとなあ、とおもっている
うまい人だったらあるもんでイイ感じビルドもできるんだろうけど、まだまだある程度は事前にこういうビルドでいきたいなあ、ぐらいの腹積もりなので、ルート運が悪いと手持ちがかなり渋いにはなりがち
エイム
スプラ3と違ってキーマウなので言い訳できねえという背水の陣
最初はダメージ0が当たり前で、とにかく当てるのがムズくて一方的にボコされるが結構あった
最近は相手の位置がわかっていて1on1で見合っていた時に、割と1マガ全弾当ててアーマー割ったら武器持ち替えてトドメを刺すとかはちょっとできるようになってきた
倍率低めのアタッチでもそれなりに当てられたりとか
しかし、足音すんなあ、からの微妙な画面外からの銃撃に向き直って当て返すがなかなか難しい
1キャラクター解禁できそうだったので、コントローラーのキャラクターがいなかったのでコースティックを解禁した
ぼちぼちあと1キャラ解禁できそうなので、シアにするかオクタンにするかを迷っている

あとコネで入るってのもね....
ていうか、このシリーズはどうなっちゃうの？ソラは軍務経験を積んでいって、ましろんは現世に帰って....ってことになるの....?

ダンス100 準決勝 感想 (ネタバレあり)
ダンサーは100人なので、コレオグラファー3人だと1人みる専の人が現れちゃうわけで、とするとここでダブルエリミネーションはほぼ確だろうということは容易に想像がつくので、それをサプライズで～すみたいな感じでクリフハンガーにするのはちょっとダセえとおもった
ブランディは出順がよかったのもあるとおもうが、堂々の決勝進出である
キーナンの方がチャレンジャー感あるかなあ
ブランディは人種的に近そう感あるので少し応援する気持ちがあったので(基本DDだけど)うれしい
マックスは初速はよかったけど、恐らくコンフォートゾーンを出るとまだまだ引き出しが厳しいのだろうなという感じがした
オチは面白かったけど、そこまでがちょっと全体的に弱かったかなあ
セリーヌはもっと前半を抑圧された、統制的なフリにして、後半はもっとしっちゃかめっちゃかにはっちゃけたフリにしたらもっと良かっただろうと素人目におもった
ていうか.....2種類も25人がポジやらフリやら全部いれてるのスゲエなと感心した、さすがはプロ
さらにエキシビジョン的に100人のコンビ芸もあったわけだし...
前半の年代別のやつみんな忘れてない？ってなっちゃった
個人的な好みかもしんないけど、60sはさすがに古すぎだなと感じるけど、70sのバイブスがマジで最高だなと
前半でも投票して、得票数の合計の方がよかったんじゃないかとおもった
あと、シリーズの最初の方はコレオグラファーとダンサー間の確執とか、ヤダミがいろいろあったけど後半になるにつれてコレオグラファー間もそれなりに大人の対応って感じのコミュニケーションだし、投票のこともあるのでとにかく平身低頭なのがなんかなと
もっとみんな本音でぶつかっていってほしいとおもった
プレッシャー・クッカーめっちゃ面白かったからなあ
プレクぐらいもっとダンサー間でダメだししあってほしい
素人目にはわからないけど、あそこのフリのところで間違えてる奴がいたとか、テンポズレてる奴がいたとか
メイキングが好きなので、後半になるにつれて練習の様子がチラっとでて、そしたらもうドレスリハーサルなのはちょっと寂しい
まあ、ダンサーたちはプロだからコレオグラファーとゴリゴリぶつかっていくとか、まとまらないとか、そういう場の雰囲気を乱すみたいなことはないのかもしれんが...

Archivebox で UA 変更
URL を追加した際に 403 ではね返されることがあった
UA で弾く系のサイトで起きてるみたい
https://docs.archivebox.io/en/v0.6.2/Configuration.html
をみると、conf を書き換えればいいようだが、web の UI に設定が反映されていないようにみえる
設定を書き換えて CLI の方では設定が反映されてたので web のサーバを再起動すればいいのかーと
ただ、 archivebox server に restart とか reload 的なサブコマンドがなくて、とりあえず process を kill した
で、過去にも archivebox server でサーバをあげてたようだが、フォアグラウンドでサーバが立ち上がってしまってシェルが抜けられなかった
起動しっぱなしにする方法がわからなかったので systemctl の service を書いてしのいだ...
