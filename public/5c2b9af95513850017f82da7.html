さっそく - diary.jgs.me

今日の午前中は踏みに行った。ジャーニー譜面踏めればいいやぐらいの気持ちで行ったら、稼働時からエクストラでソフランと低速で次々と挑戦者をいたぶっていた Monky Business が降格していたので思わず何度もプレイしてしまった。SPは楽と踊はそつなく踏める感じ、 フルコン も狙えるぐらい、激はキツかった。 DP でもついつい踏んで、楽は回転できるから楽しかったし、踊の低速地帯で音合せで渡りができるあたりすごい楽しかった。
それで、うまい うどん をすすりながら フォトカノ を消化。フォトカノ、すごいブヒ特化型 ブヒアニメ なのに、なんか悔しい面白さがあって悔しい。 写真 っていうモチーフがよかったり、画が結構綺麗めだったり、それなのに アングル が悔しいぐらいそそる感じだったりするところがいいんじゃないかとおもっている。
それから、いよいよ Reblog Monster の Windows8.1 対応をはじめた。対応といっても、 Visual Studio2013 をダウンロードしてきてプロジェクトのプロパティでTargetを Windows8 からWindows8.1に変えてやると勝手にマイグレーションしてくれるっぽくて特に何もしなかった。懸念材料だった分割画面でのサイズ変更対応は、マイグレーションするだけで CSS を書き直す必要もなかったので楽だった。あとはWIndows8系とWindows8.1系で アプリ の バイナリ を切り替えられるのかが心配。あと、8.1対応をいつリリースするのかも悩み中。やっぱり8.1の正式リリースのタイミングかなあ。
あとは、以前からずっと不安定だった ログイン まわりを問題点を思い出しながら洗ってみたりしていた。 Bootcamp の WIndows だと、輝度の刻みが大味すぎてずっとまぶしい画面を見ていなきゃいけないのが結構しんどかったりする。なんとかならないかなあ。
