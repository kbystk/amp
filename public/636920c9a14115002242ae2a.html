hammerspoon の shiftit で画面を3分割する
Shiftit から Hammerspoon に乗り換え て以来継続して使っている
デカい画面を搬入したので、up/down だけだとウィンドウがデカすぎるので3分割したくなった
少し前に Multiple window cycle sizes ていう機能が入って変数として画面サイズを投入できるようになった
spoon.ShiftIt:setWindowCyclingSizes({ 100 }, { 34 })
こういう感じのコードを足してやるだけで3分割で動くようになった
あとは
center = {{ 'cmd', 'alt', 'shift' }, 'c' }
を足して中央にもウィンドウをキーボードで移動させるようにすれば完成
それはそうと、これは macOS のユニバーサルコントロール機能を使って複数マシンを単一のキーボードで操作しているのだが、当然のようにクリップボードの共有もできていてすごい
rel: macOS のユニバーサルコントロールすごすぎワロタ
惜しむらくは、キーボードの入力が現在操作しているマシンにしか向いてなくて、キーボードでマシンをまたがってウィンドウを切り替えたい時にいちいちマウスで別マシンのウィンドウを focus しないといけないのが若干だるい
mission control でもなんでもいいのでうまい感じに切り替えられるようになってほしい
新設定の全容: https://gist.github.com/e-jigsaw/8327f5c31ac6c211e9116ee6beeae36b
