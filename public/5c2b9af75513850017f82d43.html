本日の研究室 - diary.jgs.me

今日は約72時間ぶりぐらいに 研究室 へ。今日は研究室の 環境整備 などをしましょうデイだと聞き及んでいたので、さほど期待せずに向かった。「ここが当研究室の 計算機ルーム です！ジャーン！」と案内された部屋には、うなりをあげる Pen4 の マシン 、そして ブラウン管 。マジか。
やったことといえば、 CentOS にユーザつくってもらったり、 ac.jp ドメイン の メールアドレス 発行してもらったり。ac.jpドメインは大変有用だから、増えて普通に嬉しかった。わーい。
それから、「これが当研究室 秘伝のTeXのテンプレート じゃ…」ってもらった手垢まみれの Shift_Jis の.texファイルを UTF-8 にサッと変換して久しぶりに TeX を書いてみた。 Sublime Text 2 で瞬間プレビューできるようになって僕はまた怠惰になった。でも、やっぱりTeXめんどくさいから Pandoc 試してみることに。 homebrew で Haskell-Platform をインストールして(バッテリ駆動+LTE環境で brew install はちょっと怖かった)、 cabal で pandoc をインストール。簡単なTeXだったら Markdown で十分かもしれないけれど、それなりのものを書こうと思ったら html で書いた方がいいかなあ、とおもった。生htmlはダルいから Jade で書くけど。
・・・というようなノウハウを共有したりして、徐々に研究室戦闘力を上げていきたいなあ。 シベリア と呼ばれているけれど、これからじっくりと テラフォーミング していこうという所存。
