IPoE でネットにつなぐとめっちゃ速い
引っ越し前の家での構成(OCN)
ONU(ルーター/PPPoE) => TP-Link Deco M9(ルーター) => (無線) => いくつかのデバイス
=> (有線) => いくつかのデバイス
Deco から有線もぶら下げたかったのだが、ONU に有線で直つなぎすると600Mbpsぐらい出るのが Deco の有線ポートにつなぐと10Mbpsほどまで低下してしまう減少に悩まされていた
引っ越し後の構成(IIJ) その1
ONU => TP-Link Deco M9(ルーター/PPPoE) => (無線) => いくつかのデバイス
=> (有線) => いくつかのデバイス
前の構成で2重ルーターになっていたのが問題だったのでは？と Deco にルーターを集約したが、有線ポートが遅い問題は変わらず
引っ越し後の構成(IIJ) その2
ONU => Cisco Meraki Go GX20(PPPoE) => TP-Link Deco M9(ブリッジ) => (無線) => いくつかのデバイス
=> (有線) => いくつかのデバイス
これで Meraki Go から引いてる有線はかなり速くなったが、ONU 直に比べると遅い、半分以下ぐらいになってしまう
そういえば IIJ に変えた時に IPoE なるオプションがついてたな〜とおもい、試してみることに
Meraki Go で IPoE(DS-Lite) の設定を探したのだが、なさげだった
仕方ないのでダメ元で Deco の設定を変えて試すことに
引っ越し後の構成(IIJ) その3
ONU => TP-Link Deco M9(ルーター/DS-Lite) => (無線) => いくつかのデバイス
=> (有線) => いくつかのデバイス
信じられないが、DS-Lite で接続したら Deco の有線ポートが信じられないぐらい速くなった
ONU 直とほぼ同じスピードが出る
DS-Lite だと認証書くところもなかったので、どういう仕組みになっているんだ？と謎が多いので今度調べる
