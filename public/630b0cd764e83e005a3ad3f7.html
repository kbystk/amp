セッションの日本維新の会代表選の回を聴く
自由民主党の総裁選はオファー断られたのかなんなのか扱われてなかったような気がするが、立憲民主党は公開討論会みたいなのをやってたので似たようなノリで日本維新の会の代表戦の候補者による討論会が配信されていたので聴いた
自民党の総裁選にせよなんにせよ、自分は投票権がないので全く無意味だなと虚無の気持ちにも相変わらずなるが...
前知識として、足立は無神経というかなんというかご自身の主張がおありなのはわかるが言動というかなんというか....割と嫌いだという印象、馬場は管と公開喧嘩して全く噛み合わないジジイ同士の罵り合い動画を年初あたりに拝見して今年イチ不毛な時間を過ごしたおぼえがある、梅村ははじめましてであった
トータルとしては
足立はモデレーターに割って発言することが何回かあって不快だった
国会の野次なんかもそうだけど、討論というか議論の場で勝手に喋るのはダメだとおもう
とはいえ指摘してる点はまあ納得できるが、納得してしまったらなんか負けた気になるのでしたくねえみたいな気分になってしまう
ただまあ、この回だけ聴いてると言ってることは真っ当だなあとはおもった
馬場はこの回だけでも完全に論理破綻してて、コイツ大丈夫かと...
大勢としてはコイツが人気らしいけどそれでいいのかこの会はとおもってしまう
党として危険な団体をリストアップして denylist 方式？で運用するよ～んと言いつつ、立法の話になると deny の線引きは難しいんだそうだ
ここの足立のツッコミが真っ当だなあとおもった
身から出た錆みたいな話にキレはじめるのも不快
梅村に関してはまだ議員歴もそんなに長くねえということだったが、非常に弁の立つ人なんだなあという感じは伝わったのでこの人が代表でいいんじゃないとおもった
党の姿勢としてもこういうフレッシュな人材を登用した方が通りがよいのでは？
旧統一教会の件について、足立は「公明党と創価学会の関係はある意味理想形」みたいな話をしてて、感情的にはムッとくるものの、まあそうだなとはおもった
パブリックになっていれば、支持者としてはそこも含めて投票すればいいんじゃん？ということだろうという認識
ダマでやってんのがダメなんじゃね？という指摘は自分の考えと近い
幸福実現党なんかも幸福の科学との関係を隠してるわけじゃないしなあ
ただじゃあ原則論として政教分離って？みたいな話を避けているだけのような気もする
例えば現時点では考えられない話だけど、公明党で単独政権が作れたとしてそれは宗教与党？ぽい感じになってしまうのでは？
旧統一教会についても信者数を考えれば影響は微々たるものでソーシャルインパクトは小さいという指摘はあるけど、じゃあ全く見過ごしてしまっていいのかというとうーんどうなんだろうとはおもう
あと40分ある尺の半分以上をこの話題に費やしてたけどそれでいいのかという気持ちもあった
経済の主張や外交の主張などもっと議論すべき話はたくさんあるのでは....
事件つーかテロがなければこういう話運びにはなっていなかったはずなので、重ね重ねこの件に触れ続けること自体テロの効果を強調することにしかならないとおもうんだが...
パンデミック対策についてはテクニカルなところはよくわからんし、もう疲れたというところにだけ共感した
最後に憲法改正についてで、自分は積極的に憲法はメンテナンスしていった方がいいと考えているので改憲には賛成だけど、教育無償化を撒き餌にしてついでにイロイロ変えるわ！みたいなスタンスはよくないとおもっている
で、恒例の教育無償化って憲法変えなくてもできるジャンに対してはもちろんそうだと認めつつ、法律や地方自治体の取り組みだけではキチーので政権が変わったとしても方針が継承されることを求めたいという主張
それもそうなんだろうとはおもうけど、じゃあそれだけで一旦 Pull Request 出そうや～とおもう
デカい PR 出されても困るというのをもう10年以上やってるので
夫婦別姓や同性婚なんかも憲法に書い(書き換え)ちゃえばいいジャンとおもうし、それを単発で PR 出して、それで多数をとれなかったらそういう国なんだと諦めがつく
9条をどうするだなんだみたいな話も9条だけのPR出してもらってそれで判断すればいいだけの話なんじゃないかといつもおもう
